 # Weather App

A challenge for mobile development.

## To setup the project:

 ### Go to the root folder of the project:
 Run these commands: 
 
 > npm install && npx pod-install

npm install
npx pod-install

 ios: 
 > npm run ios
 
 android: 
 > npm run android

 ## ESLint
> npx eslint

 ## Prettier
> npx prettier --write .
 
Recomendation: You should have ESLint installed globally
